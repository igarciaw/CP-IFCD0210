Etiquetas que modifican el formato del texto

Este es un texto en negrita, usando la etiqueta <strong>. El texto tiene énfasis, normalmente se usa para resaltar partes importantes.

Este es un texto en cursiva, usando la etiqueta <em>. Se usa para enfatizar el texto, lo que generalmente se muestra en cursiva.

El texto con negrita mediante la etiqueta <b> simplemente hace que el texto se vea más grueso sin implicar énfasis semántico.

El texto con cursiva usando la etiqueta <i>, que cambia la apariencia del texto, pero no necesariamente lo enfatiza de manera semántica.

Este texto tiene subrayado, utilizando la etiqueta <u>. A veces se usa para indicar enlaces, pero también puede usarse para otros fines.

Este texto está resaltado, utilizando la etiqueta <mark>. Esta etiqueta destaca el texto, como si fuera un marcador.

Este es un texto más pequeño, con la etiqueta <small>. Se usa para reducir el tamaño del texto, como en notas al pie.

Este texto está tachado, usando la etiqueta <del>. Es comúnmente utilizado para mostrar texto que ha sido eliminado o ya no es relevante.

Este texto está insertado, con la etiqueta <ins>. Se utiliza para mostrar texto que ha sido insertado, normalmente con una línea debajo.

Este es un texto en subíndice, usando la etiqueta <sub>. Se usa para crear texto en subíndice, como en fórmulas químicas.

Este es un texto en superíndice, usando la etiqueta <sup>. Se usa para crear texto en superíndice, como en notación matemática.

Este es un texto entre comillas cortas, usando la etiqueta <q>. Se utiliza para citas cortas que generalmente se muestran entre comillas.

Este es un bloque de cita
, utilizando la etiqueta <blockquote>. Se usa para citas largas, y el texto suele ir en bloque y con sangría.

Este es un texto en código, usando la etiqueta <code>. Se utiliza para mostrar texto como código, generalmente en una fuente monoespaciada.

Este es un texto en formato de teclado, usando la etiqueta <kbd>. Se utiliza para representar entradas del teclado, como si estuviera siendo escrito por el usuario.

Este es un texto de salida del programa, utilizando la etiqueta <samp>. Se utiliza para mostrar resultados o salidas de un programa.
